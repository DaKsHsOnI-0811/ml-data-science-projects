## Projects

### 1. CreditWise – Loan Approval Prediction
Predicts whether a loan will be approved based on applicant data.
Includes data cleaning, exploratory data analysis, and classification modeling.

### 2. NovaGenLab – Biomedical Data Analysis
Explores a biomedical dataset with focus on data cleaning,
type conversion, and handling missing values.

### 3. ShopSmart – E-Commerce Purchase Prediction
Analyzes online shopping behavior to predict whether a visitor
will make a purchase. Includes feature encoding and classification.

### 4. SmartCart – Customer Segmentation
Segments customers based on spending habits and demographics
using clustering techniques and feature engineering.

### Languages - Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
